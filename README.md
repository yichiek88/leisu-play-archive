# leisu-play-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能涉及交互原型、小工具、演示页面或静态内容展示，不针对任何特定域名或网站。

## 项目简介

`leisu-play-archive` 是一个轻量级的 HTML 页面存档库。主要目的是集中存放和展示一些独立的、自包含的网页文件，以便于版本管理、历史回溯和简单部署。每个 HTML 页面通常是一个完整的、不依赖外部服务的独立单元。

## 目录结构

```
leisu-play-archive/
├── index.html          # 仓库入口页面，列出所有归档页面的链接
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源目录（可选）
│   ├── css/
│   └── js/
└── README.md           # 本文件
```

- `index.html`：仓库的主页，通常包含所有归档页面的导航或列表。
- `pages/`：核心目录，每个 HTML 文件代表一个独立的归档页面。
- `assets/`：存放页面间共享的样式、脚本或其他静态资源（如存在）。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面内容不绑定任何具体域名或网站，仅供存档、展示或测试用途。
- 归档页面可能包含交互行为或动画，但一般不会收集用户数据或发起跨域请求。
- 页面可能使用外部库（如通过 CDN 加载），但会在文件内注明来源。
